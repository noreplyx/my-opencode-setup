ast-grep stdin mode evaluation
================================
Task: Find all console.log calls in piped code using ast-grep's --stdin mode

Pattern:
  console.log($$$ARGS)

Command:
  echo "const x = 1; console.log(x); const y = 2; console.log(y, 'done');" | \
    ast-grep run --stdin --lang ts -p 'console.log($$$ARGS)' --json=compact

Explanation:
  - --stdin        : Enables reading source code from standard input instead of files
  - --lang ts      : Sets language to TypeScript/JavaScript (tree-sitter grammar)
  - -p             : AST pattern to match
  - 'console.log($$$ARGS)' : Pattern matches console.log calls with any arguments.
                     $$$ARGS is a multi-argument meta-variable that captures all
                     arguments passed to console.log, including multiple comma-separated args.
  - --json=compact : Outputs results as a single-line JSON array (machine-friendly)
